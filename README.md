# Humanizing Headlines

Humanizing Headlines is a full-stack AI-powered news application designed to reduce news fatigue through intentional information design.

## Project Purpose

Modern news platforms often encourage endless scrolling, emotional overload, and information fatigue. Humanizing Headlines explores how AI and intentional UX design can create a calmer and more personalized reading experience.

This application allows users to engage with current events in a more manageable and customizable way.

---

## Features

- AI-generated article summaries
- Short / Medium / Long reading modes
- Dynamic article rendering
- Firebase authentication
- Saved article functionality
- Guest onboarding flow
- Personalized reading experience
- Responsive modern UI

---

## Technologies Used

### Frontend
- Next.js
- React
- TypeScript
- Tailwind CSS

### Backend / Services
- Firebase Authentication
- Firestore Database
- OpenAI API
- NewsAPI
- Vercel Deployment

---

## AI Integration

The application uses the OpenAI API to generate:
- Short summaries
- Medium summaries
- Long summaries

This allows users to control the depth of information they consume depending on their time, energy, or emotional bandwidth.
